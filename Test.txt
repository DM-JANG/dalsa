git Status ( 저장소의 상태를 체크하는것 ) 
git add ( 수정,삭제,추가 한 상태의 저장소를 추가하는것 ) - 깃이 다시 감시하게 하는것 
git commit ( 돌아가기 위한 지점을 잡는 명령어 ) 
git branch '이름' ( 돌아가기 위한 지점에 이름을 지정한다. )
git checkout '이름' ( 저장소를 이동하기 위한 명령어 )
git merge ( 브랜치에서  작업을 끝내고 각자 만든 모든 변경 사항을  master로 추가 )
git hush ( 자신의 로컬 commit 을 온라인으로 올린다. )
git pull ( 로컬 컴퓨터 작업 시 작업하고 있는 저장소의 최신버젼을 다운로드 )
